## 总结

```
在高密度人群中的分析存在一些局限性。
挑战包括阻塞现象、极高的杂乱程度、个体分布的不规则性、规模以及视角的不一致性。

R-CNN [2], YOLO, SSD are examples of such detectors used in recent years，....., but fails in high density crowds


The main contributions of the work are as follows:
• Crowd counting, its importance and challenges are studied.
• Various traditional and CNN based crowd counting methods are studied.

A. Traditional crowd counting approaches
Traditional approaches are classified into detection based methods, regression based methods
and density estimation based methods.

detection based methods:
...... Hence, detection-based methods can be used only for crowd counting in sparse scenes
先检测每个人，再数数量

Regression-based methods
...... To use crowd counting methods for high-density scenes, regressionbased crowd counting approach is used
不检测人，直接预测总人数

Density map estimation methods:
In cases where objects are severely occluded the
density based methods are better. It maintains some spatial
information about the crowd.
However, in these methods
features have to be manually extracted. Hence, the capability to
obtain characteristics is less accurate. Various instances are to
be manually chosen, which degrades the ability of the method.

预测一张“人群密度图”，再把它加起来

B. Convolutional Neural Network (CNN) - based methods

1) Multi-column Convolutional Neural Network (MCNN)
2) Contextual Pyramid Convolutional Neural Network (CPCNN)
3) DecideNet
4) Attention-injective


III. CONCLUSION
Low density crowd limits the traditionalmethods. 
Large density crowds can be best dealt with CNNbased approaches
```
